NanoGrabbr – первый русский бесплатный stand-alone движок для блога.

NanoGrabbr позволяет быстро и просто создавать блоги на основе разработанного нами 
бесплатного движка. Вы сможете размещать:
- видео
- тексты
- фото и другие изображения
- ссылки
- встраивать сторонние RSS/Atom-ленты

Для того, чтобы создать свой блог, вам не нужно знать PHP и основы верстки. Все, что 
вам нужно для создания блога  – это движок NanoGrabbr и хостинг с поддержкой PHP и MySQL.  
Инсталляция и настройка NanoGrabbr займет 5 минут! Просто перепишите все файлы к вам на 
хостинг и зайдите на свой сайт через браузер.

Ваш хостинг должен поддерживать PHP версии 4+ с библиотекой GDlib, Apache версии 1.3+ (или 
другой web-сервер) с поддержкой mod_rewrite и файлов .htaccess и MySQL версии 4+.

Дополнительную информацию о NanoGrabbr вы сможете найти на сайте http://nanograbbr.com

NanoGrabbr – first russian engine for standalone microblogging available for FREE!

NanoGrabbr allows you to create blogs using our free engine. Using NanoGrabbr you could post:

    * videos
    * texts
    * photos and other images
    * links
    * integrate RSS-feeds from other sites

To create your own blog, you do not need to know PHP and CSS. All you need to create a 
blog  – NanoGrabbr and hosting with PHP and MySQL support.  Install and configure NanoGrabbr 
will take 5 minutes!
